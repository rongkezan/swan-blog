Jenkins 安装

修改目录权限

将maven和jdk下载好后放入 /root/docker/目录下

因为当映射本地数据卷时，/root/docker/jenkins目录的拥有者为root用户而容器中jenkins user的uid为1000

运行 Jenkins 容器

  1. 映射端口8080到8000，50000到50000

  2. 挂载 jenkins_home 到宿主机

  3. 挂载 maven 目录

  4. 挂载 jdk11 目录

  5. 同步宿主机时间

  6. 容器内部操作宿主机shell指令

配置 Jenkins 插件镜像源

修改 /var/jenkins_home/updates/default.json文件里的镜像源地址

将 default.json 上传到服务器，使用nginx指向到该文件 https://static.51nftcard.com/update-center.json

修改/var/jenkins_home/hudson.model.UpdateCenter.xml文件里的镜像源地址

重启 jenkins
